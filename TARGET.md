# Clone Target Configuration

## Target
- **URL**: https://www.carvana.com
- **Pages to replicate**: Homepage only (marketing landing page)
- **Fidelity**: High fidelity — match layout, spacing, interactions, and visual weight. We will then iterate to apply AutoConsign's own brand identity.

## Scope — IN
- Hero section with search/CTA
- Navigation bar (structure, interactions, dropdowns)
- How it works / value proposition sections
- Featured vehicle cards grid
- Trust/proof/social proof sections
- CTA bands
- Footer structure and layout
- All hover states, transitions, and scroll animations
- Responsive behavior (desktop 1440px, tablet 768px, mobile 390px)

## Scope — OUT
- Actual Carvana branding (we will replace with AutoConsign brand tokens)
- Login/auth flows
- Search results / inventory pages (separate effort)
- Financing calculator pages
- Any pages beyond the homepage

## Post-Clone Customization Plan
After cloning, we will:
1. Replace Carvana colours with AutoConsign Vivid Teal / Warm Slate / Amber palette
2. Replace typography with Plus Jakarta Sans (headings) + Inter (body)
3. Replace hero content with AutoConsign's registration bar and seller-first messaging
4. Replace vehicle cards with AutoConsign listing card component
5. Adapt section order to match AutoConsign's homepage spec (Brand Identity v2.0 Section 7.1)
6. Replace all Carvana copy with AutoConsign voice (specific, direct, concrete, human)
